============================= установка =============================

- положить серевер в любую папку (в program files не стоит)
- в httpd.conf поменять Define SRVROOT "c:/Apache24" на путь к папке сервера
- добавить C:\apache\bin в Path
- запустить httpd в консоли

============================= настройка http/2 =============================

- в 	httpd.conf откоментировать модуль:
	#LoadModule http2_module modules/mod_http2.so
  и добавить в начале:
  	Protocols h2 h2c http/1.1

- в php.ini откоментировать расширение:
	;extension=openssl

============================= vhosts =============================

- в файле C:\Windows\System32\drivers\etc\hosts насыпать нужные домены на 127.0.0.1

	127.0.0.1         test.my
	127.0.0.1         almanac.my

- в файле httpd.conf проверить чтобы была откоментирована строка 

	Include conf/extra/httpd-vhosts.conf

- в файле httpd-vhosts.conf насыпать настройки на каждый домен

	<VirtualHost test.my>
    DocumentRoot "C:/xampp/htdocs/test/"
    ServerName test.my
		ServerAlias test.my
    <Directory "C:/xampp/htdocs/test/">
        Order allow,deny
        Allow from all
    </Directory>
	</VirtualHost>

	<VirtualHost almanac.my>
		DocumentRoot "C:/xampp/htdocs/almanac/"
		ServerName almanac.my
		ServerAlias almanac.my
    <Directory "C:/xampp/htdocs/almanac/">
        Order allow,deny
        Allow from all
    </Directory>
	</VirtualHost>

1. in /etc/apache2/sites-available/ copy 000-default.conf to something like test.my.conf (in this folder)

<VirtualHost *:80>
	ServerAdmin webmaxml@gmail.com
	ServerName test.my
	ServerAlias www.test.my
	DocumentRoot /home/webmaxml/static/work/test
	ErrorLog ${APACHE_LOG_DIR}/error.log
	CustomLog ${APACHE_LOG_DIR}/access.log combined

3. sudo a2ensite test.my.conf - this will load the test.my.conf site (will create symlink in sites-enabled folder)
4. if your DocumentRoot is not /var/www/ - add to /etc/apache2/apache2.conf something like this

<Directory /home/webmaxml/static/work/>
	Options Indexes FollowSymLinks
	AllowOverride None
	Require all granted
</Directory>

this will enable your custom folder for apache

5. add to /etc/hosts something like this

127.0.0.1        test.my

6. reload apache - sudo systemctl restart apache2
